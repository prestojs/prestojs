# Final Form

This package is used for integrating with [React Final Form](https://final-form.org/react).

It allows you to define forms quickly using a [ViewModel](doc:viewModelFactory) definition from which
the type of widget to use, the label for a field, it's help text etc can be inferred.

---

## Installation

Install the packages from npm

```bash
yarn add @prestojs/final-form final-form react-final-form
```
